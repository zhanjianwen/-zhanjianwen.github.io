1. node -v node要在4.45 以上
2. npm install -g vue-cli 全局安装 cli
3. vue init webpack sell(项目名称)
4. name: 默认 ,descrpition(描述): sell app ,Author: 回车 ,ESLint(es语法的风格检查器) : Y, Standard(标准的模板):回车, Karma+Mocha(前端单元测试的库):N, e2e test: N
5. cd sell
6. npm install(安装依赖)
7. npm run dev (运行)
